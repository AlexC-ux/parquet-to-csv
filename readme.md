# .parquet convert

Этот инструмент поможет вам конвертировать (преобразовать) файлы parquet в форматы csv или sqlite

## Как пользоваться утилитой?

### Подготовка .env

Скопируйте файл .env.template и назовите его .env

Заполните значения в файле .env в зависимости от их описания

### Генерация

Установите на компьютер `nodejs` [lts 22](https://nodejs.org/en/download)

Выполните `npm ci`

В папку `input` разместите файлы `.parquet` в любых подпапках

Выполните команду `npm run start`
